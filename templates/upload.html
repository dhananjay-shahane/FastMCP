How to Use
Upload Process
  1. Select or drag a CSV/TXT file
  2. Review the file information
  3. Click "Upload File" to upload
  4. Return to dashboard to analyze
Email Integration

Once uploaded, you can also send analysis requests via email:

  • Email requests for charts and analysis
  • Automatic processing and response
  • Results sent back with attachments
Sample CSV Format

Your CSV files should have a header row with column names. Here's an example:

date product category sales_amount quantity region
2024-01-15 Product A Electronics 1200.50 5 North
2024-01-16 Product B Clothing 850.25 12 South
